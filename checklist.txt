Release Checklist
=================

Pre-release:

- doc/CHANGELOG

Use ../ChangeLog to update changelog.


- configure.in

Update version.


- src/Makefile.am

Update LDFLAGS' version-info.


- tag it

> git tag -a libnet-x.y.z -m libnet-x.y.z
> git push --tags

Post-release:

- Post tarball to source forge.

- Announce release on libnet-dev-announce
